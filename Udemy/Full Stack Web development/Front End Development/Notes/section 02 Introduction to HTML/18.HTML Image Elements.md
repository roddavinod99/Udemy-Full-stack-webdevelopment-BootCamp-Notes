__18.HTML Image Elements__

```HTML
<img src="Vinod.png" alt= "Vinod Kumar Profile Photo">
```

img is HTML Element.

src= "angela.png" is HTML Attribute

WE can fetch a image from the internet and display it in our website or we can store it in our local storage and also display it in our website.
